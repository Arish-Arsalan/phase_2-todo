# Todo Application - Complete Implementation

This repository contains a complete implementation of the Todo application with all 5 phases:

## Phases
- **Phase 1**: Console application with Rich UI
- **Phase 2**: Foundation API & Monorepo skeleton
- **Phase 3**: AI Chatbot with MCP integration
- **Phase 4**: Local Kubernetes deployment with Helm
- **Phase 5**: Advanced cloud deployment with Kafka/Dapr

## Features
- AI-powered chatbot for natural language task management
- MCP (Model Context Protocol) integration
- Cloud-native deployment with Kubernetes and Helm
- Event-driven architecture with Kafka
- Dapr for distributed application runtime
- Advanced features: recurring tasks, due dates, priorities, tags, search, filter, sort
