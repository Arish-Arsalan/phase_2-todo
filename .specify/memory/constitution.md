# Todo Application System Constitution

## Mission
Build a spec-driven, AI-powered Todo system that evolves through 5 phases:
1. Phase I: Console app with rich UI & agent skills
2. Phase II: REST API + Monorepo skeleton
3. Phase III: AI Chatbot with MCP tools (OpenAI Agents SDK + Official MCP SDK)
4. Phase IV: Local Cloud: Minikube + Helm + Docker AI (Gordon)
5. Phase V: Production Cloud: AKS/GKE + Kafka + Dapr

## Core Principles
- No manual coding — only Claude Code + Spec-Kit Plus
- Every feature must have a Markdown spec first
- All components stateless, cloud-native, observable
- Use Agentic Dev Stack: Spec → Plan → Tasks → Claude Code

## Tech Stack
- Backend: Python 3.13 + FastAPI + SQLModel + MCP SDK
- Frontend: Next.js 14 + Tailwind + Blueprint UI (terminal/cyberpunk theme)
- AI: OpenAI Agents SDK + MCP Tools + ChatKit
- Infra: Docker + Minikube + Helm + Dapr + Kafka (Redpanda)
